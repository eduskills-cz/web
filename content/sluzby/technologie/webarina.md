# Webařina: Generování statických prezentací

Webařina není programování a zároveň to nemusí být jen skládání hotových
prvků v nějakých online nástrojích. Existují nástroje, kterým dodáte texty
v jednoduchém formátu a ony vám z toho vyrobí celý web, který už je
automaticky rychlý a bezpečný, protože nevyžaduje spouštění programového
kódu u vás ani u návštěvníka, kromě samotného prohlížeče.

Všude se mluví o frontendových a backendových webových vývojářích, ale
webová prezentace nic takového nepotřebuje. Stačí strukturovaný text, který
se promění ve výsledný barevný a především konzistentní web, kde vám
společné prvky stránek doplňuje webová šablona.

Témata:

  - Co je to web
      - Prohlížeč stáhne stránku
      - Zobrazí ji pro uživatele
      - Uživatel používá aktivní prvky
  - Co prohlížeč stahuje
      - Popis stránky v HTML
      - Vzhledové informace v CSS
      - Obrázky a další prvky
  - Čím si to ulehčujeme
      - Markdown pro texty
      - SCSS pro vzhled v CSS
      - Sestavovací nástroj
  - Vlastní webový projekt
      - Tvoříme obsah webu
      - Doplňujeme vzhled webu
      - Řešíme marketingové cíle
  - Vybrané generátory webů
      - Jekyll
      - Hugo
      - Zola
  - Co to má za výhody
      - Soustředíme se na obsah
      - Vzhled řešíme samostatně
      - Šetříme čas a nervy
  - Strukturovaná data
      - Proč je chceme mít
      - Datový formát JSON
      - Generování výstupu z dat
  - Srovnání s tiskovým světem
      - Dokumenty pro web a pro tisk
      - Publikovat PDF nebo začlenit do webu
      - Odkazy uživatele na další služby
  - Zobrazovací zařízení
      - Počítačová obrazovka
      - Chytrý telefon v ruce
  - Dobré a špatné zvyklosti
      - Srozumitelná struktura webu
      - Viditelnost aktivních prvků
      - Pohled uživatele a pohled tvůrce
